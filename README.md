README — Digital Serious Games (Web-Based)

This repository contains two web-based digital serious games designed for educational and training purposes.
The games run as standard websites and can be used immediately in a browser without installation.

The current version focuses on core gameplay functionality, while user performance saving and analytics must be implemented separately by the developer or organization deploying the game.

Features:
- Runs directly in modern web browsers
- No installation required
- Interactive gameplay mechanics
- Immediate visual feedback to users
- Modular structure that allows backend integration
- Can be embedded into websites or learning platforms
- Designed for educational and training environments
- Live Usage (Standalone Mode)

The game can run locally or from a simple web server without backend support.

In standalone mode:
- All gameplay works normally
- User progress is not saved permanently
- Performance data is lost after refresh or session end

This mode is suitable for:
- Demonstrations
- Testing
- Classroom use without tracking
- Prototype evaluation

To fully utilize the application, a backend system must be added to store:
- User identity
- Scores
- Progress
- Performance metrics
- Session history

Tested on modern browsers:
- Google Chrome
- Mozilla Firefox
- Microsoft Edge
- Safari

Recommended:
- Chrome (latest versions)

These serious games can be used in:
- Education
- Special education
- Cognitive training
- Rehabilitation
- Research studies
- Skill development programs

Current version does not include:
- Built-in user authentication
- Persistent data storage
- Multi-user tracking
- Analytics dashboards

These features must be implemented externally.
